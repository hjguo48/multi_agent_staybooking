# Week 3 Outputs Layout

This folder is organized by implementation step:

- `step1/`: Orchestrator smoke artifacts
- `step2/`: Sequential topology smoke artifacts

Each step folder contains:

- `*_state.json`: serialized `ProjectState` snapshot
- `*_report.json`: smoke check report
- `*_pipeline_report.json`: `run_experiment.py` pipeline execution report
- `*_acceptance_matrix.md`: acceptance checklist
